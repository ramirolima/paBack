Get start...
<h4>Prerequisites:</h4>

* NodeJs  
* MongoDb  

<h4>Steps run:</h4>

1. Clone the repository
2. Run MongoDb
3. Go to the project root
4. Run the command: "npm install"
5. Run the command: "npm start" or "nodemon"

And... ok, hands'on!
